# ⚡ Бот расчёта потерь электроэнергии

Telegram-бот для расчёта технических потерь электроэнергии в электросетевом хозяйстве.

**Нормативная база:** Приказ Минэнерго РФ № 326 от 30.12.2008

---

## Как работает

```
1. Пользователь загружает фото/скан однолинейной схемы
2. GPT-4o Vision распознаёт элементы схемы:
   - Марки и длины кабелей / воздушных линий
   - Марки и мощности силовых трансформаторов
3. Бот генерирует Excel-шаблон с предзаполненными данными
4. Пользователь проверяет данные, исправляет ошибки, загружает обратно
5. Бот рассчитывает константы формул (C, A, B)
6. По запросу — считает потери за любой месяц по показаниям счётчика
```

---

## Формулы (Приказ 326)

**Линии (ВЛ / КЛ):**
```
ΔW_лин = C × W²а / T

C = K²ф × k_k × r₀ × L / (U² × cos²φ × 10³)
```

**Трансформаторы:**
```
ΔW_тр = A × T + B × W²а / T

A = ΔP₀ × n            (постоянные потери, кВт)
B = ΔPк × n / (S²н × cos²φ)  (нагрузочная составляющая)
```

где:
- `W_а` — расход электроэнергии по входному счётчику (кВт·ч)
- `T` — число часов расчётного периода (месяца)
- `K²ф = 1.10`, `k_k = 0.99` — типовые значения по Приказу 326

## Требования к схеме

Для корректного распознавания схема должна содержать:
- Марки проводов/кабелей (ААБ2п 3×120, СИП 4×95, ЗАС-50 и т.д.)
- Длины участков (в метрах или км)
- Марки трансформаторов (ТМ-400/10, ТМГ-630/10 и т.д.)
- Уровни напряжения (10 кВ, 0.4 кВ)

Если что-то не указано на схеме — допиши в комментарии при загрузке.

---

## Справочные данные

Бот содержит встроенный справочник:
- Паспортные данные трансформаторов ТМ/ТМГ/ТМН (6-10 кВ)
- Удельное сопротивление r₀ для кабелей и проводов
- Автозаполнение пропущенных данных из справочника

---

*Расчёт является информационным. Для официального нормирования потерь рекомендуется верификация специалистом.*
